1. Next, Install the GitHub App

  1. Install the Streaming Slack Notify GitHub app in your desired GitHub account.
  2. Click Install
  3. Select the organization or user account to install the app
  4. Select which repositories you want to configure streaming Slack notifications for - Either all or a defined list
  5. Click Install

2. Configure GitHub App

  1. Copy the token listed above in green which will be used to link to your GitHub organization/user.
  2. On the post-install page, paste the token in the Streaming Slack App ID
  3. Specify the channel you would like all Slack notifications displayed in
  4. Optionally, you can override the default Slack Bot Username that posts the messages in the channel. (Default = GitHub Actions)
  5. Click Update to save and link your settings

3. That's it

Watch streaming logs in Slack for your workflows as you commit and send PRs to GitHub.
